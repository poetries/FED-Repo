###animation-fill-mode
**元素动画播放前后的界面显示**

![Alt text](img/1433492393840.png)
![Alt text](img/1433492470611.png)
>**none **  不改变默认行为。

>**forwards**    当动画完成后，保持最后一个属性值（在最后一个关键帧中定义）。

>**backwards**   在animation-delay 所指定的一段时间内，在动画显示之前，应用开始属性值（在第一个关键帧中定义）。

>**both**    向前和向后填充模式都被应用。
