#闭包
![Alt text](img/1434082304545.png)
![Alt text](img/1434082562151.png)

>闭包就是能够读取其他函数内部变量的函数。
>由于在Javascript语言中，只有函数内部的子函数才能读取局部变量，因此可以把闭包简单理解成“**定义在一个函数内部的函数**”。
